# Do It

Elixir Command Line Interface Framework.

A framework that helps to develop command line tools with Elixir.

## Installation

The package can be installed by adding `do_it` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:do_it, "~> 0.1.0"}
  ]
end
```
